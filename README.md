## ⌨️ Writing & Transliteration
Tools to assist with writing, vocalization, and typing in Arabic.

* [Arabizi.io](https://arabizi.io/) - A free, fully responsive virtual Arabic keyboard with a real-time phonetic transliteration engine (Yamli alternative). / *لوحة مفاتيح عربية ذكية مع ميزة التحويل الصوتي.*
* [Mishkal](https://tahadz.com/mishkal) - Arabic text vocalization and diacritization tool. / *برنامج لتشكيل النصوص العربية.*
* [Wudooh](https://wudooh.app/) - Browser extension that makes reading Arabic text easier and clearer. / *إضافة للمتصفح لتوضيح وتكبير الخط العربي.*

## 📄 Career & Productivity
Platforms to help Arab professionals build resumes, portfolios, and manage workflows.

* [Cviya](https://cviya.com/ar) - A free, AI-powered professional resume builder with full customizable layouts and native Arabic/RTL support. / *منصة مجانية بالذكاء الاصطناعي لإنشاء السير الذاتية بدعم كامل للغة العربية.*
* [Hsoub Notes](https://notes.hsoub.com/) - A simple, secure, and fast note-taking app supporting Arabic. / *تطبيق عربي لكتابة الملاحظات.*

## 💻 Development & Open Source
Libraries, APIs, and frameworks to help developers build Arabic-supported software.

* [CAMeL Tools](https://github.com/CAMeL-Lab/camel_tools) - A suite of Arabic natural language processing tools developed by NYU Abu Dhabi.
* [PyArabic](https://pypi.org/project/PyArabic/) - Python library providing basic functions to manipulate Arabic text.
* [Moment-Hijri](https://github.com/xsoh/moment-hijri) - A Hijri calendar plugin for the popular Moment.js library.
* [Quran.com API](https://quran.com/developers) - Free REST API providing Quranic text, translations, and audio.

## 🎨 Typography & Design
Premium fonts and design resources tailored for right-to-left interfaces.

* [Google Fonts (Arabic)](https://fonts.google.com/?subset=arabic) - Open-source Arabic typefaces like Cairo, Tajawal, and Almarai.
* [ArbFonts](https://arbfonts.com/) - One of the largest directories for free Arabic fonts.
* [TailwindCSS RTL](https://tailwindcss.com/docs/hover-focus-and-other-states#rtl-support) - Official documentation on how to configure Tailwind CSS for RTL (Right-to-Left) layouts.

## 🛠️ Utilities & Helpers
Fast, no-login utilities and everyday tools.

* [Hijri Converter](https://www.ummulqura.org.sa/) - Simple tool to convert Gregorian dates to standard Hijri dates.
* [Tafqeet Tool](https://tafqeet.com/) - Convert numbers and currencies into written Arabic text (التفقيط).

## 🤝 Contributing

Contributions are what make the open-source community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

1. Fork the Project.
2. Create your Feature Branch (`git checkout -b feature/AmazingTool`).
3. Add your tool to the appropriate category using the following format:
   `* [Tool Name](Link) - Short English description. / *وصف قصير بالعربية.*`
4. Commit your Changes (`git commit -m 'Add AmazingTool'`).
5. Push to the Branch (`git push origin feature/AmazingTool`).
6. Open a Pull Request.

**Rules for Submission:**
* The tool must be highly functional, free (or have a generous free tier), and natively support Arabic or RTL.
* Avoid dead links or abandoned projects.
* Keep descriptions short and punchy.
